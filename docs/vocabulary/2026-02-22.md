# Vocabulary - 2026-02-22

## continue with
- **User's version:** "go on"
- **Meaning:** A more specific and natural way to ask someone to proceed on a task already in context.
- **Example:** "Please continue with the notes based on that X post."

## as well
- **User's version:** "and the slug"
- **Meaning:** Used to add one more requested change in the same context.
- **Example:** "Please update the slug as well."
